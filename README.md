# 💰 Gold & Stock Monitor

순금 시세, 환율, 국내외 주식 가격을 실시간으로 조회하는 데스크탑 프로그램

## 기능

- **상단 Ticker Bar**: 순금(원/g), 달러·엔·유로·위안 환율 항상 표시 (60초 자동 갱신)
- **탭별 주식 검색**: 한국 / 일본 / 미국 / 유럽 탭에서 종목 코드 입력 → 오늘 가격 + 전일 대비 % 즉시 조회

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
python main.py
```

## 종목 코드 입력 방법

| 시장 | 입력 방법 | 예시 |
|------|----------|------|
| 한국 | 숫자 코드만 입력 (`.KS` / `.KQ` 자동 탐색) | `005930` → 삼성전자 |
| 일본 | 숫자 코드만 입력 (`.T` 자동 붙임) | `7203` → 토요타 |
| 미국 | 티커 그대로 | `AAPL`, `TSLA`, `MSFT` |
| 유럽 | 티커.거래소 형식 | `AIR.PA`(파리), `VOW3.DE`(독일), `SHEL.L`(런던) |
